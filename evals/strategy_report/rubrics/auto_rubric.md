# Automated Rubric Coverage

Weights follow the automated scoring chapter:

| Dimension | Weight |
| --- | ---: |
| Structure completeness | 12 |
| Source and citation quality | 18 |
| Fact and numeric accuracy | 18 |
| Strategy reasoning | 16 |
| Scenario and risk coverage | 10 |
| Chart and visual QA | 14 |
| Writing and layout | 7 |
| Compliance | 5 |

The implementation keeps the nine-module structure from the standard document. Rules cover deterministic checks; the optional LLM judge supplies professional judgment where pure parsing is weak.
