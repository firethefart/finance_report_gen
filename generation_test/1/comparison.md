# 第 0 版与第 1 版财报/策略报告对比

对比对象：

- 第 0 版网页：`generation_test/0/index.html`
- 第 1 版网页：`generation_test/1/index.html`

## 总体变化

第 0 版是一份完整的行业分析网页，已经覆盖核心结论、影响矩阵、案例卡片、风险和来源。第 1 版在此基础上，按照“策略研究报告”的正式定义和约束进行了重构：它不只是回答“并购会产生什么影响”，而是把问题拆解为政策催化、产业映射、交易样本、行业格局、情景推演、跟踪指标和合规边界。

换言之，第 0 版偏“主题解读”，第 1 版偏“正式策略研究”。

## 1. 报告定位

| 维度 | 第 0 版 | 第 1 版 |
| --- | --- | --- |
| 报告类型 | 行业专题网页 | 主题策略 / 行业格局研究 |
| 核心问题 | 北交所并购对新能源与智能电网格局有什么影响 | 并购活跃如何通过制度、产业和交易样本改变竞争边界 |
| 读者感受 | 信息清楚，适合快速阅读 | 更像券商/资管策略报告，适合研究归档和决策讨论 |
| 结论形式 | 一段式核心结论 | Investment Thesis + 执行摘要 + 反证条件 + 情景权重 |

## 2. 策略研究规范增强

第 1 版新增了策略研究中常见的正式结构：

- 报告类型、时间窗、结论属性、合规边界。
- 核心判断与反证条件。
- 执行摘要拆成 4 条 key takeaways。
- 证据池表格区分“事实、来源、策略研究含义”。
- 情景分析设置基准、上行、下行三类路径。
- 跟踪指标明确订单、利润、现金流三条验证线。
- 免责声明明确非个性化投资建议和公开资料限制。

这些改动对应 `strategy_research_framework.md` 中总结的策略研究规范：先给问题意识，再给观点层级，再落到可验证指标和风险情景。

## 3. 证据链变化

| 维度 | 第 0 版 | 第 1 版 |
| --- | --- | --- |
| 来源呈现 | 末尾来源列表 | 新增“证据池：事实、来源与分析含义”表格 |
| 事实与判断区分 | 有来源，但事实和解释混在正文中 | 每条证据单独对应一个策略含义 |
| 案例使用 | 案例卡片解释并购样本 | 案例被放入“可学习的并购逻辑”，强调交易路径、技术外溢、底层模块 |
| 可追溯性 | 较好 | 更强，读者可以看到某个结论来自哪类证据 |

## 4. 分析深度变化

第 0 版主要回答四类影响：

- 供应链控制
- 技术边界扩张
- 市场集中度
- 价格竞争缓和

第 1 版保留这些判断，但进一步拆成：

- 制度催化：并购六条、重组办法、北交所审核规则。
- 产业映射：新能源消纳、分布式接入、充换电、配网智能化。
- 交易样本：五新隧装、科达自控、旭杰科技、晶赛科技、创远信科等。
- 行业格局：细分集中度、技术资产重估、产业链补链、估值叙事切换。
- 兑现验证：订单、利润、现金流。

第 1 版的好处是读者不仅知道“会影响格局”，还知道“通过哪条路径影响、如何验证影响是否真实发生”。

## 5. 图表与表格变化

| 版块 | 第 0 版 | 第 1 版 |
| --- | --- | --- |
| 影响强度图 | 有条形图，偏视觉表达 | 保留条形图，并补充图表口径和判断依据 |
| 格局矩阵 | 影响方向、表现、含义、约束 | 升级为表现、公司动作、需验证数据 |
| 来源 | 列表形式 | 来源列表 + 证据池表格 |
| 情景分析 | 用时间线表达未来演进 | 用基准/上行/下行情景和主观概率表达 |
| 指标跟踪 | 订单、利润、现金流概念 | 每类指标下列出具体观察项 |

第 1 版更符合正式策略报告要求：图表不是装饰，而是回答明确分析问题，并包含口径说明。

## 6. 写作风格变化

第 0 版语言更像一篇清晰的行业解读；第 1 版语言更接近研究报告：

- 使用“基准情景”“反证条件”“策略研究含义”“情景权重”等研究术语。
- 避免绝对化预测，强调公开资料、研究判断和不确定性。
- 先给结论，再给证据，再给验证指标。
- 将“题材”与“基本面兑现”明确区分。

## 7. 合规与风险披露变化

第 0 版已包含“不构成投资建议”的说明，但第 1 版更完整：

- 首屏标注“合规边界：非个性化投资建议”。
- 核心判断中加入反证条件。
- 情景分析中说明概率为主观研究权重，不是统计预测。
- 页脚明确市场、政策、项目进度、交易审核和公司经营均存在不确定性。
- 强调后续应以最新正式披露为准。

## 8. 版式变化

| 维度 | 第 0 版 | 第 1 版 |
| --- | --- | --- |
| 首屏 | 视觉较强，主题清楚 | 更像策略报告封面，带报告类型和研究边界 |
| 信息密度 | 中等 | 更高 |
| 配色 | 绿色、蓝色、琥珀、玫红 | 保留克制多色，但降低装饰感 |
| 卡片使用 | 案例和指标卡较多 | 卡片服务于摘要、情景和指标，不再只是展示信息 |
| 表格 | 1 个矩阵 | 证据池表格 + 行业格局矩阵 |

## 9. 第 1 版仍可继续增强的方向

如果后续要进一步接近头部机构策略研究，可以继续补强：

- 用真实数据库补充北交所并购数量、交易金额、行业分布和审核状态。
- 加入北交所相关公司财务指标，如收入增速、毛利率、现金流和估值倍数。
- 对新能源、智能电网、工控软件、通信模块分别做细分资产图谱。
- 增加一页“投资组合映射”，例如超配/标配/低配哪些主题或因子。
- 增加可下载 PDF 版本或演示稿版本。

## 结论

第 1 版相对第 0 版的核心提升，是从“内容完整”升级为“研究规范完整”。它更明确地回答了：

- 我们为什么这么判断？
- 证据来自哪里？
- 影响通过什么机制发生？
- 哪些指标可以验证？
- 什么情况下判断会失效？
- 风险和合规边界在哪里？

因此，第 1 版更适合作为后续金融报告生成 skill 的目标样例。
